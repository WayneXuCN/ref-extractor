Reference Extractor
Extract Zotero references from Microsoft Word files

Reference Extractor is a tool to help you recover bibliographic metadata of Zotero items from Microsoft Word documents. The tool only works with Word documents stored in the .docx format. The option "Store references in document" in the document's Zotero preferences also needs to have been enabled.

The tool produces item metadata in the CSL JSON format. You can either click the "Download" button and import the downloaded JSON file into Zotero via the "Import..." option in Zotero's gear menu, or click the "Copy to Clipboard" button and use the "Import from Clipboard" option in the gear menu.

Reference Extractor is a new tool, and hasn't yet been extensively tested. If you have feedback or run into any issues, please contact me on GitHub (by filing an issue) or Twitter.
